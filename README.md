# 🎬 Movie Management System

A web-based application built using **Streamlit** that allows users to manage movie data, store records, and interact with a simple movie database.

## 🌐 Live Demo

https://movie-system-6bf5rtzzp5wvvkz47xj7wg.streamlit.app/

---

## 🚀 Features

- 🎥 Add and manage movie records  
- 📋 View stored movie details  
- 🗄️ SQLite database integration  
- 🔍 Simple and interactive UI  
- ⚡ Fast and lightweight application  

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Database:** SQLite  
- **Libraries:** Pandas  

---

## 📂 Project Structure
moviesystem.py      # Main Streamlit application
movies.db           # SQLite database (auto-created)

---

## ⚡ How to Run Locally

### 1. Clone the repository
git clone https://github.com/your-username/movie-system.git
cd movie-system

### 2. Install dependencies
python3 -m pip install streamlit pandas

### 3. Run the app
streamlit run moviesystem.py
